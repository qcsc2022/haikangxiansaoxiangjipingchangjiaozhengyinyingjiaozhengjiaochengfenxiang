## 平场校正的必要性

一般情况下，以下原因需要进行平场校正：

1. **光照不均**：光照在图像的不同区域分布不均匀，导致图像亮度不一致。
2. **镜头中心和镜头边缘响应度不一致**：镜头中心和边缘的成像效果可能存在差异，影响图像的整体质量。
3. **成像元器件的各像元响应不一致**：相机传感器上的各个像元对光线的响应可能存在差异，导致图像出现不均匀的亮度分布。
4. **固定的图像背景噪声**：图像中可能存在固定的背景噪声，影响图像的清晰度和质量。
